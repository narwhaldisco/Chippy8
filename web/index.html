Chippy8 is a Chip8 emulator/interpreter written in javascript by Nick Ward

Check out the source on github

Keys are mapped in a 4x4 grid starting at 1-4 and ending at Z-V.

1 2 3 4
Q W E R
A S D F
Z X C V

Controls vary from game to game. I suggest button mashing.

Good one to start with is Invaders: W to start, then Q moves left, E moves right and W shoots
